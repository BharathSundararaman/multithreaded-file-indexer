# Multithreaded File Indexer

## Description
A multithreaded Java application that performs content-based indexing of text files using an inverted index.

## Features
- Content-based indexing
- Fast keyword search
- Multithreaded file processing

## Concepts Used
- Java Collections
- OOP Design
- ExecutorService
- ConcurrentHashMap
- Regex Processing

## How to Run
1. Place .txt files inside testfiles folder
2. Run Main.java
3. Enter search queries
